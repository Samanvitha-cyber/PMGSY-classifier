# PMGSY classifier
## Problem Statement
Problem Statement

Intelligent Classification of Rural Infrastructure Projects

The Challenge

The Pradhan Mantri Gram Sadak Yojana (PMGSY) is a flagship rural development program in India aimed at providing all-weather road connectivity to eligible unconnected habitations. Over time, the program has expanded through multiple schemes such as PMGSY-I, PMGSY-II, RCPLWEA, and others, each with distinct objectives and project characteristics.

Managing and classifying thousands of rural infrastructure projects manually is a complex and time-consuming task for government agencies and infrastructure planners. Manual classification can lead to inconsistencies, errors, and difficulties in monitoring project performance and resource allocation.

Proposed Solution

This project develops a Machine Learning-based classification system that automatically predicts the appropriate PMGSY scheme for a road or bridge construction project based on its physical and financial attributes.

The solution leverages IBM watsonx.ai AutoAI and an XGBoost Classifier model to improve classification accuracy, reduce manual effort, and support data-driven decision-making in rural infrastructure planning.

Dataset

Pradhan Mantri Gram Sadak Yojana (PMGSY) Dataset from AI Kosh.

Dataset Source:
https://aikosh.indiaai.gov.in/web/datasets/details/pradhan_mantri_gram_sadak_yojna_pmgsy.html
